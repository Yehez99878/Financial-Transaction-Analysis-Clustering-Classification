Financial Transaction Analysis: Customer Profiling & ClassificationProyek ini adalah submission akhir untuk kelas "Belajar Machine Learning untuk Pemula" di Dicoding. Fokus utama proyek ini adalah melakukan segmentasi perilaku transaksi nasabah menggunakan teknik Clustering dan membangun model prediksi melalui Classification.📌 Deskripsi ProyekDataset yang digunakan mencakup 2.512 sampel transaksi dengan berbagai atribut seperti demografi nasabah, pola penggunaan, dan saldo akun. Proyek ini mensimulasikan kebutuhan bisnis untuk memahami profil nasabah guna mendukung keamanan finansial dan pengembangan model prediktif.Alur Kerja (Workflow)Data Preprocessing & Cleaning:Menangani missing values menggunakan dropna() dan menghapus data duplikat.Melakukan feature encoding menggunakan LabelEncoder untuk data kategorikal.Melakukan feature scaling menggunakan StandardScaler agar fitur numerik memiliki rentang yang seragam.(Advanced) Penanganan outlier menggunakan metode IQR (Interquartile Range).Unsupervised Learning (Clustering):Menggunakan K-Means Clustering untuk mengelompokkan nasabah.Menentukan jumlah cluster terbaik ($k=2$) melalui Elbow Method dengan metrik Silhouette Score.Visualisasi hasil pengelompokan dalam ruang 2D menggunakan PCA (Principal Component Analysis).Supervised Learning (Classification):Membangun model untuk memprediksi label cluster yang telah dihasilkan.Membandingkan performa model Decision Tree dan Random Forest.Melakukan optimasi model melalui Hyperparameter Tuning menggunakan GridSearchCV.📊 Hasil AnalisisBerdasarkan hasil clustering, nasabah terbagi menjadi dua profil utama:Cluster 0 (Nasabah Senior Konservatif): Memiliki rata-rata usia dan saldo akun yang lebih tinggi, namun dengan nilai transaksi yang cenderung lebih rendah.Cluster 1 (Nasabah Muda Aktif): Memiliki rata-rata usia lebih muda dan saldo lebih rendah, namun menunjukkan aktivitas transaksi yang sedikit lebih tinggi.Performa model klasifikasi mencapai akurasi maksimal (1.00) pada tahap pengujian data (test set sebesar 20%).🛠️ Teknologi & LibraryPython: Bahasa pemrograman utama.Pandas & NumPy: Manipulasi dan analisis data.Scikit-Learn: Implementasi model Machine Learning dan preprocessing.Matplotlib & Seaborn: Visualisasi data dan matriks korelasi.Joblib: Untuk menyimpan dan memuat model yang telah dilatih (.h5).Yehezkiel Athanasius Manurung (Anju)Informatics Student at Gunadarma University | AI Engineer Trainee
